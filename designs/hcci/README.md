# Homogeneous Charge Compression Ignition (HCCI) Chamber Design

This directory contains simulation input files and data plots for an HCCI combustion chamber.

## Files
- `simulation_input.json`: Simulation parameters for the HCCI chamber.
- `performance_plot.png`: Plot showing pressure-volume diagram and heat release rate.
- `emissions_data.csv`: NOx, CO, and HC emissions data across load range.
