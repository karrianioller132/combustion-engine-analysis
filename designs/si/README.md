# Spark Ignition (SI) Chamber Design

This directory contains simulation input files and data plots for a traditional Spark Ignition combustion chamber.

## Files
- `simulation_input.json`: Simulation parameters for the SI chamber.
- `performance_plot.png`: Plot showing pressure-volume diagram and heat release rate.
- `emissions_data.csv`: NOx, CO, and HC emissions data across load range.

*Note: These are placeholder files for documentation purposes.*
